# NeetCode 150 Practice Repository

Hi! 👋  

This repository is where I document my journey through the **NeetCode 150** challenges. 
The goal is not only to solve problems from **LeetCode**, but also to keep track of my **learning process**, notes, and code experiments.

---

## Repository Structure
To keep learning and problem-solving organized, the repo is structured using **folders by topic**.
Each topic contains two subfolders:

- `intro/` — for notes, explanations, and code snippets while learning the concepts.  
- `leetcode/` — for implementing solutions to the LeetCode problems related to that topic.  

This structure helps me track **both the learning process and problem-solving practice** side by side in an organized way.
